# Rust Learning Projects

This repository contains various Rust projects built for learning and experimentation.

## Projects Included

* **[csvreader-rust](./csvreader-rust/)**: A command-line utility for reading and processing CSV files.
* **[decompres-rust](./decompres-rust/)**: A command-line utility for decompressing files.

## Getting Started

To run any of these projects, ensure you have [Rust and Cargo installed](https://www.rust-lang.org/tools/install). 

1. Clone the repository:
   ```bash
   git clone git@github.com:yaffalhakim1/rust-learning.git
   ```
2. Navigate into the desired project directory:
   ```bash
   cd rust-learning/csvreader-rust
   # OR
   cd rust-learning/decompres-rust
   ```
3. Run the project using Cargo:
   ```bash
   cargo run
   ```
